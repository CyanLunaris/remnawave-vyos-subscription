# remnawave-vyos-subscription

Синхронизирует VPN-ноды с панели [Remnawave](https://github.com/remnawave) на VyOS, запускает sing-box как прозрачный TUN-прокси с автоматической ротацией нод.

**Поддерживаемые протоколы:** VLESS · VMess · Trojan

---

## Быстрый старт

### 1. Создать config.env

```bash
mkdir -p /config/remnaproxy /config/remnaproxy/logs /config/sing-box
```

`/config/remnaproxy/config.env`:
```bash
SUBSCRIPTION_URL=https://panel.example.com/sub/TOKEN
```

Остальные параметры — см. [config.env.example](config.env.example).

### 2. Запустить контейнер

```
set container name remnaproxy image 'ghcr.io/cyanlunaris/remnawave-vyos-subscription:latest'
set container name remnaproxy capability net-admin
set container name remnaproxy allow-host-networks
set container name remnaproxy device tun source '/dev/net/tun'
set container name remnaproxy device tun destination '/dev/net/tun'
set container name remnaproxy volume config source '/config/remnaproxy'
set container name remnaproxy volume config destination '/etc/remnaproxy'
set container name remnaproxy volume singbox source '/config/sing-box'
set container name remnaproxy volume singbox destination '/etc/sing-box'
set container name remnaproxy volume logs source '/config/remnaproxy/logs'
set container name remnaproxy volume logs destination '/var/log/remnaproxy'

commit ; save
```

### 3. Установить TUI

```bash
sudo bash container-setup.sh
```

Устанавливает `/config/scripts/remnaproxy-tui` (переживает обновления VyOS) и симлинк в `/usr/local/bin`.

---

## TUI

```bash
remnaproxy-tui
```

| Клавиша | Действие |
|---------|----------|
| `F2` | Список нод, переключение (↑↓ + Enter) |
| `F3` | Редактор конфига |
| `s` | Форс-синк |
| `q` | Выход |

---

## Конфигурация

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| `SUBSCRIPTION_URL` | — | Ссылка-подписка (**обязательно**) |
| `SYNC_INTERVAL` | `10min` | Интервал синхронизации |
| `HEARTBEAT_INTERVAL` | `30s` | Интервал проверки связи |
| `HEARTBEAT_FAIL_THRESHOLD` | `2` | Неудач подряд до смены ноды |
| `HEARTBEAT_HOST` | `cp.cloudflare.com` | Хост для проверки связи |
| `GEO_DIRECT_IP` | `private,ru` | GeoIP-коды прямого маршрута |
| `GEO_DIRECT_SITE` | `category-ru` | GeoSite-коды прямого маршрута |
| `TUN_INTERFACE` | `tun0` | Имя TUN-интерфейса |
| `TUN_ADDRESS` | `172.19.0.1/30` | Адрес TUN-интерфейса |
| `RULE_SET_DIR` | `/etc/sing-box` | Директория для `.srs` rule-set файлов |

## Обслуживание

**Обновить образ:**
```bash
sudo podman pull ghcr.io/cyanlunaris/remnawave-vyos-subscription:latest
sudo systemctl restart vyos-container-remnaproxy
```

**Логи:**
```bash
journalctl -u vyos-container-remnaproxy -f
tail -f /config/remnaproxy/logs/sync.log
tail -f /config/remnaproxy/logs/heartbeat.log
```

**Статус:**
```bash
sudo systemctl status vyos-container-remnaproxy
```
